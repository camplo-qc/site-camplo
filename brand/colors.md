# Camplo — tokens de marque

## Couleurs

| Token | Hex | Usage |
|---|---|---|
| `--fern` | `#3E6B31` | Couleur principale (sapin, texte, surfaces foncées) |
| `--marmalade` | `#E8853B` | Accent (tente, CTA chaleureux, call-outs) |
| `--paper` | `#F6F0E0` | Fond crème neutre |

## Typographie

- **Wordmark / titres** : Cabinet Grotesk 800
- **UI / body** : Satoshi 500 / 700
- **Secondaire / labels** : General Sans 500 / 600

Fontshare : https://api.fontshare.com/v2/css?f[]=cabinet-grotesk@800,700,500&f[]=satoshi@900,700,500&f[]=general-sans@600,500&display=swap
